# Découpage retenu

## Talk 1 — Types applicatifs

Objectif : aider l’équipe à choisir entre `class`, `record class`, `struct`, `record struct` et `readonly record struct` dans du code métier.

Les exemples principaux sont `HashSet`, `Dictionary`, `IEqualityComparer`, clés mutables, IDs typés et value objects.

## Talk 2 — Types mémoire

Objectif : approfondir les types orientés performance et mémoire : `Span<T>`, `Memory<T>`, `ref struct`, `stackalloc`, `MemoryPool<T>` et les restrictions de lifetime.

Ce découpage évite de transformer une présentation de 30 minutes en survol de 45 minutes.
